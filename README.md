# \# SOC Analyst Academy: End-to-End Blue Team Operations Portfolio

# 

# This repository is a structured, artifact-driven simulation of enterprise Security Operations Center (SOC) environments. It contains deep-dive incident investigations, behavioral detection rules, and proactive threat hunting playbooks targeting modern Windows threat vectors.

# 

# \---

# 

# \## 🏗️ Repository Architecture

# 

# \* `📂 Incidents/` — 10 detailed technical investigations containing root cause analysis, timeline construction, and analyst notes.

# \* `📂 Detections/` — Custom behavioral detection logic, Sigma/SIEM rules, and mitigation strategies.

# \* `📂 Threat-Hunts/` — Hypothesis-driven hunting queries and log analysis playbooks.

# \* `📂 IOC-Repository/` — Centralized database of malicious IPs, domains, URLs, and file hashes.

# \* `📂 Playbooks/` — Step-by-step incident response workflows aligned with industry frameworks.
